📋 How to get your Razorpay keys
Follow these steps to connect your Razorpay account
  1. 1

    Create a Razorpay account

    Sign up at dashboard.razorpay.com and complete KYC verification for live payments.

  2. 2

    Get your API keys

    Go to Settings → API Keys → Generate Key. Copy both the Key ID and Key Secret.

  3. 3

    Paste keys above

    Use rzp_test_ keys while testing. Switch to rzp_live_ when going live.

  4. 4

    Set up Webhook (optional but recommended)

    In Razorpay Dashboard → Settings → Webhooks, add:
    https://apiv2.aasaitech.in/webhooks/razorpay
    Select events: payment.captured, payment.failed, refund.processed

  5. 5

    Enable payments & save

    Toggle Enable Payments on, click Test Connection, then Save Settings. Your storefront checkout is live!
