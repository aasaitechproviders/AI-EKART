How to get your API keys
Follow these steps to connect your Razorpay account.
  1. 1

    Create a Razorpay account

    Sign up at dashboard.razorpay.com and complete KYC verification for live payments.

  2. 2

    Get your API keys

    Navigate to Settings → API Keys → Generate Key. Copy both the Key ID and Key Secret immediately — the secret is shown only once.

  3. 3

    Paste keys above

    Use rzp_test_ keys while testing your integration, then switch to rzp_live_ when going live.

  4. 4

    Set up Webhook (recommended)

    In Razorpay Dashboard → Settings → Webhooks, add:
    https://apiv2.aasaitech.in/webhooks/razorpay
    Events: payment.captured, payment.failed, refund.processed

  5. 5

    Test & go live

    Toggle Enable Payments on, click Test Connection, then Save Settings. You're live!
